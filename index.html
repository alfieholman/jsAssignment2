Different ways of implementing CSS

there are 3 different methods for accessing CSS these are Inline, Embedded and External bellow I will describe what they are and give some examples on them to make it easier to understand.

The first of the three is Inline this is what I considers the most simple of the three methods this uses the style to edit attributes but only on one element in html this means that it doesn't cover the whole web page just that just that one instance for example if you have a embedded CSS that added a style to every paragraph but you wanted one paragraph to be completely different you could edit that one paragraph in a Inline CSS this would overrule the embedded CSS rule but that would be only in this one instance which is very important not to mistake it for the whole web page.

The next one is Embedded this is slightly different to inline it still uses the style to change attributes but instead of only editing only one element it edits the whole web page it does this by having all the edited attributes in the head part of the html this will effect the whole document unless inline CSS is used anywhere on the page this will overrule the edit element. this is in my opinion the most import type of CSS because it can affect your whole web page and can change the whole feel of a website quite easily.

The final way of accessing CSS is External this is also done in the head of html but instead of writing all of the edited elements and attributes you instead link the web page to a file which contains all of the styles that you want in your document this means that if you have a large website with lots of pages then you don't need to go through each one and change the embedded CSS you can instead link all your pages to that file and then edit the file which will reflect on every page that is linked to that file making updating a large website much easier.

Box model in html

The box model is a key part of understanding how html works you can think if it as a box with many boxes inside it each smaller than the one before the one being on the edge is the margin then the border, padding and then content in the middle each part holds different things in html. The content part is the simplest to understand it is the part where words, images or videos are shown to the user. Next is the padding area this is what goes around the edge of the content it can be edited with padding top, padding left, padding right, padding bottom as well as just padding. next is the border area this goes around the padding this will create a gap between the padding and the margin .The margin is a transparent gap that goes all the way around the edge of the content you can edit the size of the margin different elements in the code.
